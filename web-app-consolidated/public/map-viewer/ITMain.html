Asset Registration

Click the "Asset" button in the sidebar to open the registration form. Note: You must fill all fields (ID, Name, Type) to successfully create the asset in the database.

Decommissioning

To remove an asset, click its icon on the map to open the Slide-in Panel. Click the Red Delete Button. Warning: This action is permanent and removes all history.

Backend Maintenance

Operational Task Standard Procedure
Server Initialization Verify Node.js is running on Port 3000. Without the backend active, map pins and history logs will not populate.
Data Volatility The System Data Model is currently in-memory. Avoid restarting the server unless necessary, as it clears all current asset history.
Hardware Check If the "Last Seen" timestamp stops updating, verify the BLE Gateway power and check the tracker battery.
